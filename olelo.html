‘Ōlelo Hawai‘i Base Site

Base Site

I learned the process of creating a base site from scratch. By creating a base site from scratch I learned how to use HTML, CSS, and Javascript to code a website. Learning how to use HTML to house content, CSS to style the webpage, and Javascript to make it interactive/functional.

Final

I filled the base site with resources to make learning ‘ōlelo Hawai‘i more accessible. I styled the site with the fonts Optima for body copy and Kabel STD for headings giving the site a more Hawaiian style.
